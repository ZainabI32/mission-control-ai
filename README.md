
Purpose:
Managers often spend too much time digging through long updates, guessing team morale, and reacting to problems too late. AI Mission Control provides a single, clear dashboard that turns raw text into actionable intelligence.

Future Enhancements
-Automated 24‑hour action plan
-Meeting agenda generator
-Weekly report export
-Elevated user interface



Set up and running:
Install dependencies
Code:
pip install flask
pip install groq
pip install python-dotenv

Start the backend
Code:python app.py

Open the dashboard
Visit: http://127.0.0.1:5000
